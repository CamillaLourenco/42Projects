### Pré-requisitos

Este projeto requer os seguintes softwares:
- GCC
- Make
- uma máquina virtual (como Virtualbox)
- Sistema operacional Unix

### Instalação

Clone o repositório para sua máquina local:
```sh
git clone xxxxx.git

```
